Value

Depending on the function prefix: If the check is successful, all functions return TRUE. If the check is not successful, assertAtomicVector/assert_atomic_vector throws an error message, testAtomicVector/test_atomic_vector returns FALSE, and checkAtomicVector returns a string with the error message. The function expect_atomic_vector will return an expectation.

Description

An atomic vector is defined slightly different from specifications in is.atomic and is.vector: An atomic vector is either logical, integer, numeric, complex, character or raw and can have any attributes. I.e., a factor is an atomic vector, but NULL is not. In short, this is equivalent to is.atomic(x) && !is.null(x).
